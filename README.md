# Round 2 – Coding Puzzle Challenge

## Problem 1 – Grid Rotation
- Rotated rows based on directions from `directions.txt`.
- Took the middle row and summed ASCII values.
- Output: 385 → Hex: 181.

## Problem 2 – String Processing
- Reversed string, removed every 3rd character.
- Shifted ASCII +2, counted vowels.
- Output: 4.

## Problem 3 – State Machine
- Checked tokens in `states.txt`.
- Counted how many start and end with the same character.
- Output: 5.

## Final Key
